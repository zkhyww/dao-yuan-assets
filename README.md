# Dao Yuan Assets

Static visual asset package for Dao Yuan / Daoist Roots.

Canonical deploy payload is the `public/` directory.

After this repository is pushed and GitHub Pages is enabled with the included workflow, assets are intended to be referenced like:

```text
https://<username>.github.io/dao-yuan-assets/home-backgrounds/01-home-dao-yuan.webp
```

No business source code is included in this package.
